Create New User

Creating a new user is crucial because you should avoid managing your full node under the root user account. Our user will be named sentinel, and you will be asked to create a new password for this user.

adduser sentinel

Grant sudo access to sentinel user. Open the sudoers file

nano /etc/sudoers

Add the following line

/etc/sudoers
sentinel ALL=(ALL:ALL) ALL

Switch to the newly created user

su - sentinel
